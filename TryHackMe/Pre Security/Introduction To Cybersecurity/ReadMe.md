## Cybersecurity Careers - 06-02-2026
- Security Analyst - Maintain and monitor the security of organization's data
- Security Engineer - Build, design, and fix security systems
- Incident Responder - First responder ( computer firefighters) to cyber attacks to reduce or stop them and protect systems
- Digital Forensics Examiner - Uses digital forensics to investigate incidents and crimes
- Malware Analyst - Analyse types of malware to know what they are and how they work
- Penetration Tester - Test the security systems and software of a company for loopholes, flaws, and vulnerabilities
- Red Teamer - Stimulate real-world, adversarial attacks on organization's digital systems, physical premises, and employees to identify vulnerabilities and test defensive readiness
